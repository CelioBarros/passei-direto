# passei-direto

### Enter in MYSQL and execute
```
  source {path-to-project}passei-direto/passei-direto-back/sql/database.sql
```

### Install and execute backend
```
cd passei-direto-back
npm i
node index.js
```

### Install and execute frontend
```
cd passei-direto-front
npm i
npm run serve
```
